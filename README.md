# Forense AI

Plataforma multi-forense com IA — pesquisa em bases de dados forenses, ingestão de evidências, análise com IA, geração de relatórios periciais e gestão de casos.

> Stack: Python + FastAPI · PostgreSQL + RLS · SQLAlchemy 2.0 · Next.js 14 + TypeScript

## Estrutura

```
.specify/          # SPEC-DRIVEN — specs, planos, constituição, research
  memory/          # constituição + feature log
  specs/           # specs por feature
  research/        # referências GitHub e dados de treino
```
